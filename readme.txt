Adds UAE emirates (subdivisions) to WooCommerce checkout and hides the flat rate option when free shipping is available.

== Description ==

UAE WooCommerce Suite is a lightweight plugin that enhances WooCommerce for businesses operating in the United Arab Emirates.  
It automatically adds the 7 Emirates (Dubai, Abu Dhabi, Sharjah, Ajman, Fujairah, Ras Al Khaimah, Umm Al Quwain) to the checkout “State” list, and hides the Flat Rate shipping option whenever Free Shipping is available.

**Features**
* Adds all 7 Emirates as WooCommerce states.
* Automatically hides Flat Rate when Free Shipping is available.
* Lightweight and compatible with all WooCommerce themes.
* Optimized for UAE-based stores.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/uae-woocommerce-suite` directory, or install the plugin through the WordPress Plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Enjoy UAE-ready WooCommerce checkout functionality.
